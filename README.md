   
## Анна Ляписова | QA Manual Engineer 

### ❓ Почему я в QA?
* Люблю разбирать системы по кирпичикам
* Не жду идеальной документации — начинаю тестировать сразу, выявляю требования в процессе и фиксирую их для команды
* Верю, что качество — это про заботу о пользователе

### 🛠 Ключевые навыки:
* Ручное тестирование веб‑ и мобильных приложений
* Составление тест‑кейсов и чек‑листов
* Документирование дефектов
* Базовое взаимодействие с API
* Работа с инструментами разработчика
* Основы SQL

* Внимательность к деталям
